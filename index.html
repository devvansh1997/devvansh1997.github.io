Research

Publications and ongoing research in computer vision and multimodal reasoning.

MaRIO: Multi-agent Collaborative Reasoning
in MLLMs

Introduced MaRIO, a benchmark evaluating how multimodal LLMs reason across independent agent observations of a shared environment — without shared camera poses or metadata. Evaluated 23 state-of-the-art MLLMs across 3,255 QA samples. All models fall substantially below human performance (73.3%), with Gemini-2.5-Pro reaching 56.7% as the best model.

Submitted to ECCV 2026

BGVP: Object Detector Comparison
on Pedestrian Safety

Evaluated YOLO-X, YOLOv5, YOLOv4, Faster R-CNN, and EfficientDet on a vulnerable pedestrian dataset. Analyzed detection performance under occlusion and low-light conditions relevant to autonomous driving safety.

Published at IEEE ITSC 2024

Real-time Workout Activity Recognition
Fellowship.AI

Built an end-to-end CV pipeline achieving 96.7% classification accuracy. Integrated pose estimation (MediaPipe/BlazePose) with temporal modeling (RepNet) to handle noisy real-world inputs and enable robust activity classification and repetition counting in dynamic environments.

Projects

End-to-end computer vision systems built across segmentation, recognition, and medical imaging.

  • SAM BBox Automation

    Automated instance segmentation pipeline comparing Grounding DINO vs. YOLOv11x as detection backends, both feeding into SAM2. Evaluated on CamVid — YOLOv11 achieves Dice 0.63, outperforming DINO by 49%.

  • CLIP on HAR

    Fine-tuned CLIP and SigLIP on a 15-class activity recognition dataset using contrastive learning with cross-batch memory (XBM). SigLIP achieves F1 0.86 and 86.1% accuracy with a 32k effective batch size.

  • Brain Tumor Segmentation

    2D U-Net pipeline for segmenting tumor regions from 3D BraTS MRI volumes. Combined Dice + CrossEntropy loss, 5-fold cross-validation, evaluated with Dice score and HD95. Designed for local and HPC reproducibility.

  • TinyML Affordance Recognition

    Designed and deployed a TinyML-based vision system for real-time affordance detection on an Arduino Nano BLE Sense. Optimized a CNN for on-device inference under strict memory and compute constraints.

  • HRNet Pose Estimation

    End-to-end HRNet-w48 inference pipeline without MMCV. Reimplemented affine transforms, heatmap decoding, and skeleton visualization from scratch to demonstrate the full inference pipeline transparently.

  • Otsu Thresholding

    From-scratch implementation of Otsu's automatic image thresholding algorithm. Maximizes between-class variance across 256 threshold values to find the optimal binary segmentation without any manual tuning.
